### What I learned

I learnt another use of css flexbox

```css
.cardContainer {
        display: flex;
        height: 70vh;
    }
    .container>h2{
        line-height: 1.5;
        margin-bottom: 1rem;
        font-size: 2.5rem;
    }
    .header{
        font-size: 2.7rem;
    }
    .topParagraph{
        padding-left: 27rem;
        padding-right: 27rem;
        margin-bottom: 2rem;
    }
    .cardContainer p{
        font-size: 1.2rem;
    }
    .card1{
        width: 33%;
        margin-left: 2rem;
        height: 48%;
        margin-top: auto;
        margin-bottom: auto;
    }
    .section{
        width: 33%;
        margin: 0px 3rem;
        height: 100%;
    }
    .card4{
        width: 33%;
        height: 48%;
        margin-right: 2rem;
        margin-top: auto;
        margin-bottom: auto;
    }
```
